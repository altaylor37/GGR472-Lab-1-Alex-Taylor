Please find below a GeoJSON file containing a marked up version of the UofT campus. There are three examples of points, lines, and polygons, all of which have been classified using colors and symbols to represent similar groups.
GeoJSON file of UofT Campus

These classifications can be best seen in the metadata of each object in the GeoJSON file.

Points: Blue = UofT Building; Red = TTC Station
Lines: Thick = Avenue; Thin = Street
Polygons: Grey = UofT Building; Green = Park
